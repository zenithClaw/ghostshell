# GhostShell

**GhostShell: The invisible interface between your screen and your thoughts.**

GhostShell is a macOS desktop context layer that provides an ultra-low latency, always-on-top, frosted-glass scratchpad. It lives independently of your desktop workspaces and follows your active monitor.

## Features
- **Global Shortcut**: Press `⌥ Option + Space` anywhere, on any desktop, over any fullscreen app to summon the GhostShell.
- **True Frosted Glass**: Built with macOS native vibrancy logic through Electron. Not a fake transparent window, but an authentic translucent blur.
- **Context-Aware**: If you have multiple monitors, GhostShell summons exactly where your cursor is.
- **Zero Friction**: No title bars, no borders. Just a floating canvas for your thoughts. 

## Run Locally
1. `npm install`
2. `npm start`
